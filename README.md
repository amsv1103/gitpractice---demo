## Project Overview
This repository is created to practice Git and GitHub workflows.
I used this project to understand repository initialization,
branching, merging, and undo/reset operations.

---

## Git Workflow Used
- Initialized a local Git repository
- Staged and committed changes
- Connected the repository to GitHub
- Pushed changes to the remote repository

## Commands used
- git init
- git add .
- git commit -m "message"
- git push origin main

---

## Branching and Merge
- Created a feature branch to work independently
- Made changes in the feature branch
- Switched back to the main branch
- Merged the feature branch into main
- Pushed the updated main branch to GitHub

## Commands used
- git checkout -b feature-branch
- git checkout main
- git merge feature-branch
- git push origin main

---

## Undo and Commit History
- Viewed commit history
- Undid the most recent commit when required

## Commands used
- git log
- git reset HEAD~1
